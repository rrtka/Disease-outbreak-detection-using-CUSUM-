# Disease Outbreak Detection using CUSUM 

This project uses statistical methods to detect disease outbreaks from weekly flu (ILI) data by identifying abnormal increases using CUSUM control charts.

##  Tech Stack
- Python
- Pandas
- Matplotlib

##  Features
- Data preprocessing & cleaning
- Weekly flu trend analysis
- CUSUM algorithm implementation
- Outbreak signal detection
- Visualization of anomalies

##  Output
- Flu trend visualization
- Highlighted outbreak signals
- Total outbreak count
